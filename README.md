# What is CorruptedAI?

CorruptedAI is an attempt to build a **GPT-style Large Language Model completely from scratch** using Python and PyTorch.

No APIs.

No wrappers.

No copy-pasting mysterious repositories from GitHub and pretending they're mine.

Every major component will be built manually, understood completely, and probably rewritten three times because I realize I misunderstood linear algebra.

# Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CorruptedAI.git
```

Move into the project

```bash
cd CorruptedAI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run something

```bash
python LLM/tokenizer/demo.py
```

Hope for the best.
